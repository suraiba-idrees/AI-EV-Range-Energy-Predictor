# 🏎️ NEURA-DRIVE: AI-Powered EV Engine

**Advanced Electric Vehicle Range & Energy Consumption Predictor**

### 🌟 Project Overview
Neura-Drive ek intelligent dashboard hai jo machine learning (Random Forest & XGBoost Ensemble) ka istemal karte hue EV ki range aur energy consumption predict karta hai.

### 🧠 Key Features
* **Real-time Telemetry Analysis**: Speed, weight, aur weather ka asar dekhein.
* **Regenerative Braking Logic**: Energy recovery ka asar range par.
* **XAI (Explainable AI)**: SHAP plots ke zariye samjhein ke AI ne ye faisla kyun liya.

### 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **ML Models:** Scikit-learn (Random Forest), XGBoost
- **Analytics:** SHAP, Plotly

### 🚀 How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the dashboard: `streamlit run dashboard.py`
